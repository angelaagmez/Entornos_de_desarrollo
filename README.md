# Entornos_de_desarrollo

## Descripcion
El programa "Hola mundo" simplemente muestra "Hola mundo" por pantalla.

## Compilacion y ejecucion del programa
Para **compilar** el programa teclea lo siguiente:

```console
javac Holamundo.java
```

Y para **ejecutarlo**:

```console
java Holamundo
```
